# roles/

Per-project roles. Most projects inherit the standard Coder role set
defined in [`../../system/roles/`](../../system/roles/) and only need to
record **deviations** here:

- A role that's customized for this project (e.g. project-specific
  permissions, extra tools).
- A role that's *omitted* for this project.
- A role that's added beyond the standard set.

If you're using the standard set as-is, this folder can stay empty
except for this README.
